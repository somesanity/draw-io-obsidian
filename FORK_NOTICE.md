### Modifications Made

Version 2.1.0 includes the following enhancements developed with AI assistance:

1. **Extended File Format Support**
   - Added support for `.drawid` files
   - Improved `.drawio.svg` file handling
   - Right-click menu option to open any SVG with draw.io

2. **Performance Improvements**
   - Implemented iframe caching for 10x faster loading
   - Optimized server startup with health check polling
   - Reduced diagram switching delays

3. **Bug Fixes**
   - Fixed `.drawio` files being saved as SVG format
   - Fixed editor title not updating
   - Fixed close button not responding
   - Fixed file corruption issues

4. **UX Enhancements**
   - Better tab management (temporary/pinned tabs)
   - Custom theme-aware file icons
   - Improved context menus
   - Create diagrams from folder menu
